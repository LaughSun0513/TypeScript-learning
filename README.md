# TS的 安装 编译 vscode自动编译
```
npm i typescript -g
tsc index.ts

# vscode 
tsc --init       // 生成tsconfig.json
"outDir": "./js" // 修改outDir路径
vscode终端 --> 运行任务 ---> tsc:监视
```

# ts中的数据类型

- 字符串(String)
- 数字类型(Number)
- 数组类型(Array)
- 元组类型(Tuple)
- 枚举类型(Enum)
- 任意类型(Any)
- void类型
- never类型
- null / undefined
